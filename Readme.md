# README #

An Initializer point for cmake project


### pre-requisite ###

* gcc
* cmake
* ninja

[exp-windows-10]
* Install scoop from https://scoop.sh/
* scoop install gcc
* scoop install cmake
* scoop install ninja


### how-to-build ###

* mkdir build
* cd build
* cmake .. -G "Ninja"
* ninja


### how-to-execute ###

* .\build\application\application.exe
